PACMAN AI

In this project, I have learned to implement PACMAN AI to avoid or eat ghots/food pellets in the game. The project started off very simple with just PACMAN finding food pellets using search algorithms such as DFS, BFS, Dijkstra and A*. As the game gets more complex with ghots, minimax and expectimax, logic and bayes net are used to choose the best moves. Finally, I used reinforcement learning especially Q-learning and approximate Q-learning to allow pacman to learn by itself the best moves by repeatedly playing the game. PACMAN AI plays this game better than me eventually :).

SPAM OR NOT

In this project, I separated data into 2 sets, training set and validation set. Using only a list of words (feature) such as ('opportunity', 'win', 'easy', 'low', 'free') to train the logistic regression model on. Then use this model to test on data that it has not seen before in the test set. The dataset is a real-world dataset from SpamAssasin(https://spamassassin.apache.org/old/publiccorpus/). This classifier has the accuracy of over 93% in classifying spam from non-spam emails.
